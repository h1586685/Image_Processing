# Image_Processing
Foundations of Digital Image Processing

## Feature :
>File
- Load / Save image
- Image Info

>Setting
- Get ROI
- Warp images:
1. Rotation
2. Translate
3. Affine transform
4. Perspective transform

- Show image histograms
-  Change the image color space including:
1. RGB
2. BGR
3. HSV
4. GRAYSCALE

>Image Processing
- Thresholding
- Histogram Equalization
- Filter
- Morphological Transformations including:
1. Erosion_img
2. Dilation_img
3. Open
4. Close

>Edge Detections
- Canny 
- Sobel
